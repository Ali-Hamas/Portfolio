# Ali's Portfolio

A cyberpunk-themed portfolio website showcasing AI Engineering and Backend Infrastructure expertise.

## 🚀 Features

- **Cyberpunk/Terminal Aesthetic**: Dark mode design with neon accents
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Smooth Animations**: Interactive UI with scroll-based effects
- **Performance Optimized**: Fast loading and smooth interactions

## 🛠️ Technologies

- HTML5
- CSS3 (Custom Design System)
- Vanilla JavaScript
- Vercel (Deployment)

## 📦 Deployment

This portfolio is deployed on Vercel for blazing-fast performance and global CDN distribution.

### Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML/CSS/JS!

### Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Deploy with one click
