# Project Images Folder

This folder contains all project screenshots and images for the portfolio.

## How to Add Projects:

1. **Add your project images** to this folder
   - Recommended size: 1200x800px or similar 3:2 ratio
   - Formats: PNG, JPG, or WebP
   - Name format: `project-name.png` (use lowercase with hyphens)

2. **Example naming:**
   - `ai-chatbot.png`
   - `ecommerce-website.png`
   - `automation-tool.png`

3. **Image Guidelines:**
   - High quality (but optimized for web)
   - Clear showcase of your work
   - Professional screenshots or mockups

## Current Project Images:

The portfolio currently shows 2 placeholder projects. Once you add your real project images here, update the HTML to reference them.

### To update project images in the portfolio:

In `index.html`, find the project cards and update the `src` attribute:

```html
<img src="images/projects/your-project-name.png" alt="Your Project Name">
```
